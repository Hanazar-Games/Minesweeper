# 超级扫雷 - 更新日志

## v1.3.0 (2026-04-27)

### 新增功能
- **♾️ 无尽模式**  
  关卡制无限扫雷：每关胜利后自动进入下一层，难度随层数递增（棋盘变大、雷密度提升）。踩雷扣血（非立即死亡），血量随时间缓慢回复，每揭示20格额外回1血。每5关提升最大血量上限。首次点击设为重生点，新关卡从此开始。

### 优化与调整
- **🎮 默认经典扫雷**  
  "开始游戏"按钮现在直接进入经典初级（9×9/10雷），不再需要先选择难度，降低新玩家上手门槛。
- **🔘 UI/UX 比例修复**  
  全面调整菜单按钮、游戏棋盘和 HUD 的尺寸比例，在高 DPI 屏幕和移动端上显示更协调。

## v1.0.1 (2026-04-27)

### 修复与优化
- **🛡️ 启动画面安全机制增强**  
  在 `index.html` 的 `<head>` 中内联紧急后备脚本：若 5 秒内启动画面仍未消失，强制通过 `style.display = 'none'` 移除遮罩层，确保即使外部 JS 加载失败也不会卡住主界面。

- **🔧 Script 加载顺序修复**  
  将 `solver.js` 从 `game.js` 之后移至 `game.js` 之前加载，消除 `Game.hint()` 调用 `Solver.getHint()` 时的潜在时序风险。

- **🧹 全局代码审查与防护**  
  对所有初始化模块（`Settings.load()`、`UI.init()`、`Stats.getAll()` 等）添加 try-catch 错误隔离；为 `getElementById` 调用添加 null/undefined 守卫；确保任何单点失败不会阻断整体启动流程。

- **🌐 浏览器兼容性改进**  
  全面移除生产代码中的可选链操作符 (`?.`)，恢复为传统的 `&&` 短路求值或 `if (el)` 判空，兼容 IE11 / 旧版 Edge 等不支持 ES2020 的环境。

### 已知限制
- 浏览器控制台可能出现 `localStorage is not defined` 或 `document is not defined` 的 Node.js 语法检查警告，这些属于浏览器全局 API，在真实浏览器环境中正常运行，并非 Bug。

## v1.0.0 (2026-04-27)

### 核心功能
- 经典扫雷游戏引擎：5 种难度 + 自定义尺寸
- 种子系统：通过种子码分享和复玩同一地图
- 每日挑战：基于日期的固定种子 Intermediate 地图
- 无猜模式：首次点击 3×3 安全区保障
- AI 求解器：约束传播 + 概率计算的智能提示

### UI 与体验
- 10 屏 SPA 架构（启动画面、主菜单、难度选择、游戏、教程、设置、统计、排行榜、挑战、帮助）
- 9 标签页专业设置面板（样式 / 语言 / 音频 / 动画 / 性能 / 快捷键 / 其他 / 关于）
- 6 步交互式教程，带练习棋盘和智能提示
- 完整的键盘导航支持（WASD、方向键、Space、Enter、F、C、H、P、Ctrl+Z）

### 音频与特效
- 纯 Web Audio API 程序化合成：10 种 SFX 风格 + 10 种音乐风格
- ADSR 包络、混响、节奏控制等高级音频参数
- 粒子效果、平滑动画、页面过渡

### 数据与社交
- 成就系统：7 大类 35 个成就，带 Toast 通知
- 统计与排行榜：按难度追踪，含图表展示
